Definición

Acciones que no podemos ver y pasan dentro de una web



Guarda y maneja información



Un ejemplo puede ser facebook



Para su desarrollo se utilizan distintos lenguajes



Conceptos

Claves

Clientes

Envían solicitudes
al back-end

Servidor

Computadora que recibe las solicitudes, aunque cualquier computadora que esté conectada a una red puede actuar como un servidor

Aplicación

Se ejecuta en el servidor que escucha las solicitudes, recupera información y envía una respuesta

Bases de datos

Proporcionan una interfaz para guardar datos de forma persistente en la memoria

¿Cómo funciona?

Servidor

Recibe una solicitud en forma de un localizador de recursos uniforme (URL) y obtiene la información necesaria

HTTPS o HTTP

Protocolo
Indicador de si la solicitud está o no encriptada
No afecta la manera en que el servidor maneja la solicitud

Dominio

Host
No afecta la manera en que el servidor maneja la solicitud
A dónde debe llegar la respuesta

Recurso o ruta

Path
Qué quiere el usuario y encuentra la sección específica que debe ser ejecutada para obtener la respuesta

Parámetro

Query string
Se usa en la sección para responder a la solicitud adecuadamente

Métodos HTTP

Información obtenida por el URL se debe combinar con método de petición que define qué subdivisiones, pueden ser: get, post, put o delete.

Acción + recurso

Encuentra sección y con base en los parámetros da respuesta
Normalmente página html

Servidor

Barrera entre mundo exterior y todas las partes de la web
Accesible a través de rutas específicas

  • Estados Unidos


    Estos números son un promedio entre los salarios de todos los usuarios en puestos de "Back-end developer". El rango de "Most likely range" representa los valores que caen entre los percentiles 25% y 75% de toda la data presente de este puesto laboral.
    Según los datos dados por los usuarios, en el caso de Estados Unidos el salario promedio de un desarrollador de Back-end no tiene variaciones significantes dependiendo de los años de experiencia, y las compañías más populares son IBM y Google ambas siendo empresas centradas en creación de nuevas tecnologías.

    Source

  • Canadá


    Estos números están en dólares canadienses, por lo que en dólares estadounidenses el salario promedio en Canadá para puestos de "Back-end developer" es $57,845.
    En el caso de Canadá, las empresas más populares son Shopify y IBM, siendo Shopify una empreza de comercio global mientras que IBM es una empresa centrada en tecnología.

    Source

  • Colombia


    Estos números están en pesos colombianos al mes, por lo que al año serian $132M de pesos, o $32,000 dólares estadounidenses.
    En el caso de Colombia las empresas más populares son Rappi y Globant, siendo Rappi una empresa de comercio digital y Globant una empresa que provee servicios de soporte en tecnologias de la información.

    Source


  • Características Laborales


    Se encuentran desafíos que es neceario resolver con la mayor eficiencia posible, tales como:
    • Responder a las solicitudes de los clientes obteniendo, procesando y enviando los datos requeridos.
    • Optimizar el código y el sistema para poder manejar un tráfico pesado y múltiples solicitudes.
    • Verificar y autentificar las solicitudes validas.
    • Organizar multiples servidores en la nube.
    Para lograrlo se requiere planear con anticipación qué lenguaje de programacion se usará, cada lenguaje tiene ventajas y desventajas, por lo que es importante escoger el lenguaje que mejor se adapte a las especificaciones para tu pagina web.
